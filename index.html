Mapping Earthquakes

This project is a showcase of my skill in calling API's with JavaScript. The webpage I've created is a map of Earthquake data from the past 7 days called from a GeoJSON API provided by the United States Geological Survey. The map is called from a MapBox API, and the layering is provided by Leaflet. The Earthquake's are displayed by circles in which their size correlates to the magnitude of the Earthquake. Additionally each circle includes popup information which relays the location and exact magnitude.

Closing Price Prediction

This is the final project that I delivered as part of my Data Analytics course at UC Davis. It is a flask application connected to MongoDB, and hosted on Heroku. The web app employs a supervised machine learning algorithm to predict the closing price of a stock as being either a gain or a loss for the day. The user has the ability to enter in a desired asset that they would like to predict and customize the algorithm as well as the training period that they would like the machine to train with.
